# glovewort

Design-first repository for a lightweight deployment pattern based on:

- **OpenClaw 2026.4.2** as the hand
- **Ollama** as the brain
- **FastAPI bridge** as the model adapter
- **Telegram** as command ingress

This repository is currently in the architecture/specification phase.
